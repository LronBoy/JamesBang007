JS SDK for OAuth2.0 http://open.weibo.com/wiki/index.php/Weibo-JS_V2 开发指南

1、网站接入绑定域名

为了保证你的应用或网站的安全，在使用Weibo JS SDK时，需要你在应用信息页面为该应用选定绑定域名功能，并绑定你的域名，我们将以此域名来判定Weibo JS SDK API请求来源的合法性，不合法的请求将被拒绝。

2、增加命名空间

在您页面的HTML标签中增加XML命名空间。 < html xmlns:wb="http://open.weibo.com/wb" >

3、引用JS SDK文件

在你的页面部署wb.js，同时，如果你的页面编码不是UTF-8，请添加charset="utf-8"属性。 如果你的应用处于开发调试过程，建议使用JSSDK的调试模式，开启调试模式，只需增加一个参数debug=true即可，JSSDK会在浏览器的控制台或自带的JS调试工具中输出相关的日志信息，以方便调试。
